## peças de uma linguagem 

- comentarios 
- declaração de variáveis
- operadores (atribuição, concatenação, matemáticos, lógicos)
- tipos de dados (string, number, array)
- controle de fluxo (if/else)
- estrutura de repetição (for, while)

acresentar os dados
